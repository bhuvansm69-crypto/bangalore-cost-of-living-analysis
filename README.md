## 📌 Overview

The **Bangalore Cost of Living Analysis Dashboard** is an end-to-end data analytics project that explores the cost of living across **50 Bangalore localities** using **100,000 synthetic records**.

The project covers the complete analytics workflow:

- Data Cleaning
- Data Analysis
- SQL Analytics
- Interactive Dashboard
- Geospatial Visualization
- Business Insights

Built using **Python**, **SQLite**, **Pandas**, **Plotly Dash**, and **Bootstrap**, this dashboard allows users to explore housing, groceries, transportation, fuel, utilities, internet, gyms, and food delivery costs through a modern responsive interface.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/bhuvansm69-crypto/bangalore-cost-of-living-analysis.git
```

Move into the project

```bash
cd bangalore-cost-of-living-analysis
```

Create a virtual environment

```bash
python -m venv venv
```

Activate it

Windows

```bash
venv\Scripts\activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the dashboard

```bash
python dashboard/app.py
```

Open

```
http://127.0.0.1:8050
```
