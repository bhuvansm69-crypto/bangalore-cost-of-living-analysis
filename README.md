## 📌 Overview

This project explores the cost of living across Bangalore by analyzing over **100,000 records** spanning rent, groceries, transportation, fuel, utilities, gyms, internet services, and food delivery.

The dashboard enables users to compare localities, providers, pricing trends, and affordability through interactive analytics.

## ▶️ Installation

Clone the repository

```bash
git clone https://github.com/bhuvansm69-crypto/bangalore-cost-of-living-analysis.git
```

Navigate to the project

```bash
cd bangalore-cost-of-living-analysis
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the dashboard

```bash
python dashboard/app.py
```

Open:

```
http://127.0.0.1:8050
```
