## Overview

Bangalore is India's Silicon Valley — a city where the cost of living varies dramatically across neighborhoods. From premium rents in Koramangala to budget-friendly options in Electronic City, understanding these patterns matters for residents, expats, and businesses.

This dashboard transforms raw data into actionable insights through interactive visualizations built on a complete analytics pipeline:

```
┌─────────────┐    ┌─────────────────┐    ┌─────────────┐
│  Data Layer │───▶│ Analytics Layer │───▶│ Visual Layer│
│  (SQLite)   │    │ (Pandas/NumPy)  │    │(Plotly Dash)│
└─────────────┘    └─────────────────┘    └─────────────┘
```

## SQL Analytics

The project uses SQLite for efficient storage and analytical queries.

### Cost of Living Index

```sql
WITH category_averages AS (
    SELECT
        locality,
        category,
        AVG(cost) as avg_cost,
        STDDEV(cost) as cost_stddev
    FROM cost_of_living
    GROUP BY locality, category
),
locality_totals AS (
    SELECT
        locality,
        SUM(avg_cost * weight) as weighted_cost
    FROM category_averages
    JOIN category_weights USING (category)
    GROUP BY locality
)
SELECT
    locality,
    weighted_cost,
    RANK() OVER (ORDER BY weighted_cost DESC) as cost_rank
FROM locality_totals;
```

### Top Expensive Localities

```sql
SELECT
    locality,
    ROUND(AVG(cost), 2) as avg_cost,
    COUNT(*) as record_count,
    ROUND(MIN(cost), 2) as min_cost,
    ROUND(MAX(cost), 2) as max_cost
FROM cost_of_living
GROUP BY locality
ORDER BY avg_cost DESC
LIMIT 10;
```

### Monthly Trends

```sql
SELECT
    strftime('%Y-%m', date) as month,
    category,
    ROUND(AVG(cost), 2) as avg_monthly_cost,
    COUNT(*) as transactions
FROM cost_of_living
WHERE date >= date('now', '-12 months')
GROUP BY month, category
ORDER BY month DESC, avg_monthly_cost DESC;
```

### Provider Comparison

```sql
SELECT
    provider,
    category,
    ROUND(AVG(cost), 2) as avg_cost,
    ROUND(AVG(rating), 1) as avg_rating,
    COUNT(*) as service_count
FROM cost_of_living
GROUP BY provider, category
HAVING service_count > 50
ORDER BY avg_cost ASC;
```

> All SQL scripts are in the `sql/` directory and can be run independently for custom analysis.
